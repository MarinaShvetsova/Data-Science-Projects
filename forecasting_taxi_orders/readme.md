# Описание проекта - Модель прогнозирования количества заказов такси на следующий час
## Данные

Даны исторические данные о количестве заказов такси в аэропортах.

## Задача

Разработать модели предсказания объемов заказа такси на следующий час. Значение метрики RMSE на тестовой выборке должно быть не больше 48.

## Используемые инструменты и библиотеки
*Python*, *Pandas*, *Scikit-learn*, *временные ряды*, *машинное обучение*
